# Oldway Barbershop Website

Премиальный сайт для барбершопа с системой онлайн-записи и панелью администратора.

## Стек технологий

- **Frontend**: Next.js 15 (App Router), Tailwind CSS, Framer Motion
- **Backend**: Next.js API Routes
- **Database**: SQLite с Prisma ORM
- **Testing**: Vitest

## Особенности

- **Публичный сайт**:
  - Современный адаптивный дизайн в темных тонах.
  - Плавные анимации при прокрутке.
  - 4-шаговая система онлайн-записи.
  - Автоматическая генерация ссылки на WhatsApp для подтверждения записи.
- **Админ-панель** (/admin):
  - Защищенный вход (пароль: `oldway_barber321`).
  - Управление барберами (добавление, редактирование, удаление).
  - Управление услугами и гибкое ценообразование (своя цена для каждого мастера).
  - Управление расписанием (открытие/блокировка временных слотов).
  - Просмотр и управление записями.

## Установка и запуск

1. Клонируйте репозиторий
2. Установите зависимости: npm install
3. Настройте переменные окружения в .env
4. Настройте базу данных: npx prisma db push
5. Запустите проект: npm run dev
